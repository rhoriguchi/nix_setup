# nix-setup

## Switch to unstable package channel

```bash
sudo nix-channel --add https://nixos.org/channels/nixos-unstable nixos
```

## Nixops

### Init

```bash
nixops create nixops.nix
```

### Info

```bash
nixops info
```

### Deploy

```bash
sudo nix-channel --update
nixops deploy
```
